# 中证1000 时序择时(v4 定稿)

遗传规划(GP)公式化因子挖掘 + walk-forward 时序择时。**单标的指数多空择时,非选股策略。**

**成绩(2018-01-01 至 2026-07,双边万5):年化 ~27% / 夏普 ~1.3 / 最大回撤 −17.8% / 卡玛 ~1.5**;同期持有 512100:4% / 0.19 / −46.3%。

---

## 策略流程速览

```
6指数日频OHLCV ──> GP逐季挖因子(rank IC) ──> 因子信号化(40日rank) ──> cos IC加权合成
                                                                        │
实盘/回测仓位 <── tanh(Φ⁻¹(π)) 映射 <── 组合信号40日分位 π <── (+熊态增强ΔS) ──┘
```

### 2. 因子挖掘

**因子 = 表达式树**(公式化因子,Kakushadze《101 Formulaic Alphas》范式),例:`ts_std(ts_sum(abs(returns),3),5)`。

**零件库(定稿 26 算子口径):**
- 双目:`add sub mul div`(除法防零)
- 单目:`neg abs sign ssqrt slog`(符号平方根/符号对数,保号防爆)
- 滚动统计:`ts_mean ts_std ts_min ts_max ts_sum ts_rank ts_zscore`
- 时序结构:`delay delta ema ts_corr ts_slope ts_argmax ts_argmin ts_decay_linear ts_er(Kaufman效率比) ts_tsi(真实强度指数)`
- 窗口参数:`{3, 5, 10, 20, 40, 60, 120, 244}`
- 叶子(9):`open high low close volume amount returns entropy120 beta120`
- 注:代码库另含 `ts_ac1/ts_cj/ts_vr5` 三个后加算子,**定稿库由 `mine_ops26` 摘除后挖出**,复现请用该入口

**GP 参数**:种群 200 × 20 代;锦标赛 k=5;交叉 0.7 / 变异 0.25 / 其余复制;ramped half-and-half 初始化(深度 2–4);深度上限 6;复杂度罚 λ=0.001×节点数;训练段覆盖率 ≥70%;逐(季度×库)确定性随机种子。

**两级库结构**:每季度挖 **7 个库** = 1 个类库(6 指数联合面板,IC 取 6 列平均)+ 6 个专属库(单指数)。中证1000 交易信号只用「类库 ∪ 专属 zz1000」。

**统计指标与治理(全部阈值,预注册跑后不改):**

| 环节 | 指标 | 阈值 |
|---|---|---|
| GP 适应度 | \|train rank IC\| − 复杂度罚 | — |
| 名人堂(HOF) | train/valid rank IC 同号且双达标;库内 Spearman 相关 | ≥0.02 / ≥0.02;≤0.7;容量 20 |

### 3. 信号组合

1. **单因子信号化**:`S_i = (rank40(因子值) × 2 − 1) × sign_i`,S_i ∈ [−1,1];sign 为录取时按 train IC 定的方向号(IC 为负反着用)
2. **cos IC 定权**(每季度末,季内锁定):先把 S_i 过一遍生产同款映射得单因子仓位 `p_i = tanh(Φ⁻¹(rank40(S_i)))`,再算过去 12 个月
   `cosIC_i = Σ(p_i·r_{t+1}) / √(Σp_i² · Σr_{t+1}²)` ——**分子即该因子单独交易的盈亏**;权重 `w_i = max(cosIC_i, 0)`,黄牌 ×0.5
3. **凸组合**:`comb = Σ w_i·S_i / Σ w_i`(当前在役约 88 个 GA 因子;13 个人工"元老"因子明文剔除——其方向 2013–14 预热期锁死,已失效)
4. **熊态增强**:iVIX/shibor/北向构造熊市概率,熊态时压反转/波动族、抬动量族(家族再倾斜),产生修正量 ΔS 叠加到 comb;贡献约 +0.8pp,非熊态 ΔS=0

> **为什么挖矿用 rank IC、加权用 cos IC**:挖掘要抗噪的方向筛选(秩统计每日等票,不被极端日骗,GP 骗不了);加权要对齐盈亏的计量(cos 分子即钱)。同一把尺子用两遍:cos挖+cos权 过拟合(实测夏普 0.84),rank权 丢强度信息(1.01);分工使用 = 1.37。

### 4. 策略构建与持仓

**不是个股组合——持仓是单一 ETF 的一个连续仓位数:**

```
π_t    = rank40(comb+ΔS)   同质参照系:每季用本季权重回算[季前60交易日,季末]的
                            comb 序列作排位参照,窗口内不混新旧权重体制
仓位_t = tanh(Φ⁻¹(π_t)) ∈ (−1,1)    π 先做 −0.5/40 连续性修正并 clip 到 [1e-6, 1−1e-6]
```

- **多空口径(定稿)**:仓位 ∈ [−1,1],负值做空(空头腿以 IM 中证1000 股指期货实现);**纯多口径**:clip 到 [0,1],看空即平仓吃货基
- 信号 T 日收盘生成,**T+1 生效**(结算 `p.shift(1)`);调仓成本 `|Δ仓位|×0.0005`(双边万5);组合信号=0 或标的无行情日强制空仓
- 实测:均仓位 ~60%,日换手 ~30%,满仓(|仓|>0.9)约占 1/4 时间

### 5. 无前视与防过拟合

三段数据完全隔离(train 8年 / validation 2年 / 成绩段) · 逐季 walk-forward 只用当时可见数据 · 全部滚动窗口只回看 · 权重窗口末端剔 1 日 · 入库三关 + 黄牌退役 + 复职限一次 · 阈值预注册 · 存档重放须先逐日复现引擎原值(<1e-12)方可采信。
